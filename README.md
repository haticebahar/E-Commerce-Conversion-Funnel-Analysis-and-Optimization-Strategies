# E-Commerce-Conversion-Funnel-Analysis-and-Optimization-Strategies
120.000 kullanıcı oturumu üzerinden Python ve Plotly ile e-ticaret dönüşüm hunisi analizi. Ödeme süreçlerindeki sürtünme (friction) noktalarının teşhisi, cihaz bazlı performans segmentasyonu ve %50'lik checkout terk oranını minimize edecek veri odaklı stratejiler. Ürün yönetimi (PO/BA) bakış açısıyla ROI artırıcı çözüm önerileri.
